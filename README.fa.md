# breeze

[![Build Status][travis-badge]][travis-link]
[![Slack Room][slack-badge]][slack-link]

ابزار میانبر برای دستور git در پوسته ماهی.

این از [SCM Breeze] (https://github.com/scmbreeze/scm_breeze "SCM Breeze") الهام گرفته شده است.
